Project STUKOV

STUKOV is an smart tool, wich can be used for "drive" a RC car, it can seem weird, but it's work !
Inside, we can found some components, and any one have a specifical function. All of the 3d's plans used are availables with THIS LINK

This tool is an bionic arm, inside, we've got some components, wich are listed here :

  • an accelerometer, because the car is piloted with this component. It's used for drive, for go forward, you tilt STUKOV forward too, same for backward, left and righ direction.
  • an Arduino card, it's the motherbord which serve at link every components between them
  • a radio module, because I wouldn't had a linked car, and this module is compatible with the Arduino card

And it's all, I wasn't wanting to have a too fat arm, I was wanting the skinnyer as possible, I have reduce the number of components in this objectiv.
The fist problem when I was wanting to create this, it's the modelisation, because, we don't realized when we havn't try, but it's so hard ! I swear, because, like you can see in the 3d plans, it's a lot of bended surfaces, and I havn't enought skills for did it whith iron or metals, so I've done it with a 3d printer (thanks Wanahoo [unsponsored content] and my brother for the help). The harder isn't the modelisation, but the code, in fact, learn Arduino's code wasn't easy at the begening, because it's very different than others languages I've learned before, but it's one of my favorite part, I'm not an engineer, I'm a programer.
